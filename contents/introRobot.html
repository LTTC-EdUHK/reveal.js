    Disciplines related to Robotics
  • Specific robot is designed for some tasks, no universal robot
  • Development of robot involves multiple disciplines across fields:
    • Engineering: robot's system including hardware and software
    • Mechanics: kinematics, motion feedback
    • Sensing and signal processing: convert environmental information to data
    • Networking: Ability to communicate with other devices or Internet
    • Robotic algorithms: artificial intelligence, machinge learning, path finding, cognition
    • Operating system and software engineering: integrate system firmwares, hardware drivers and user application into robot
  • Possible challenges when developing a robot:
    • Mechanics: study of kinematics, sensor/motor feedback
    • Signal processing: covert sensor raw data into to program's object data
    • Mathematics: investigate fundamental problems related to the robot's task
    • Robot algorithms: optimize computation for robotic tasks
